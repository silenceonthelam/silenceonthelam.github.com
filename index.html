Projects

Most of these projects are works-in-progress and so-far only work in newer webkit browsers.

  • TUMP

    TUMP is a synthesizer created with the Web Audio API. It allows the user to select from four types of oscillators and four temperaments. It has 88 keys and a slider device for switching octaves on more narrow monitors. There is also a volume control and a tuning control.

    Upcoming Changes

    • UI Redesign
    • More responsive UI
    • UI Browser Compatibility
    • Custom Timbres

  • PhoEdit

    Dynamically adjust various aspects of a photo, such as opacity, saturation, and blur with CSS filters.

    Upcoming Changes

    • More responsive UI
    • UI Browser Compatibility

  • Web Audio Phone

    The Web Audio Phone is a smaller project that uses the Web Audio API to create tones with DTMF.
